# 🤖 Mood AI Agent

A personality-based AI chatbot built using **LangChain**, **Mistral AI**, and **Streamlit**.

## Features

* 😡 Angry AI Mode
* 😂 Funny AI Mode
* 😢 Sad AI Mode
* 💬 Conversation Memory
* 🎨 Streamlit Chat Interface

## Tech Stack

* Python
* LangChain
* Mistral AI
* Streamlit
* python-dotenv

## Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
MISTRAL_API_KEY=YOUR_API_KEY
```

Run the application:

```bash
streamlit run streamlit_chatbot.py
```
